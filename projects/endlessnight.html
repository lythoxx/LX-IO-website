Endless Night

Endless Night is 2D side-scroller fantasy RPG using a medieval setting. It is being developed in Godot and will be available to Windows Linux and MacOS.

The game is currently in development and will be available for download soon.

A mobile version is not planned.

The game will feature Celeste-like movement as well as combat against multiple different enemies, a story and self-composed music. The game is composed of multiple different levels, each a few minutes length. It is planned to be about 4-5 hours of gameplay.
